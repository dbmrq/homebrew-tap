# Homebrew Tap

This is a [Homebrew](https://brew.sh) tap for my personal tools.

## Installation

```bash
brew tap dbmrq/tap
```

## Available Formulae

### Ralph

AI-powered task automation tool.

```bash
brew install dbmrq/tap/ralph
```

Or install directly:

```bash
brew install dbmrq/tap/ralph
```

## Updating

```bash
brew update
brew upgrade ralph
```
